# StayNova

# Hotel Reservation System

A Java console application for managing hotel room bookings and cancellations.

## Features
- Book rooms
- Cancel reservations
- View room availability
- Simple room management
- Object-Oriented implementation

## Technologies
- Java
- OOP
- Arrays

## Project Structure
```
StayNova/
│
├── StayNova.java
├── README.md

```

## How to Run
1. Compile the project
2. Run `StayNova.java`
3. Use the menu to manage bookings
